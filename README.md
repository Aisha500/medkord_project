# MedKord

**Your Medical Records, Always Accessible.**

MedKord is a digital healthcare solution built specifically for the Nigerian healthcare system. It digitizes patient consultation history, enabling healthcare providers to record information faster while allowing patients to securely access their medical records from any device.

## Live Demo
Experience the live application here: [https://medkord.lovable.app](https://medkord.lovable.app)

## Key Features
* **Digitized Patient History:** Replaces vulnerable paper records with a secure digital system.
* **Provider Efficiency:** Streamlined interface designed for healthcare professionals to log consultations quickly and accurately.
* **Patient Accessibility:** Empowers patients by allowing them to view their medical history on the go from any web-enabled device.
* **Responsive Design:** Fully optimized for seamless use across mobile phones, tablets, and desktop computers.

## Technologies Used
